Method, in one paragraph

Behavioral pathologies are classified by nine deterministic, pure-function scorers, each producing a score in [0, 1] plus the concrete sessionIds and tradeIds that drove the score. A priority-ordered tiebreaker handles score collisions — session_tilt intentionally loses ties because it is a downstream consequence of any pathology that produces losses. The control trader is correctly classified by a CONTROL_THRESHOLD of 0.5: when no signal crosses it, the prediction is control. Two pathologies (FOMO vs plan_non_adherence) initially shared signal; they were separated cleanly using rationale-text features that appear in only one trader each. The full reproduction command is python -m eval.run_eval.
